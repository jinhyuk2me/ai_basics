## 수식 작성 규칙

### 기본 문법
- 인라인 수식: `$...$`
- 블록 수식: `$$...$$` (들여쓰기 없이 왼쪽 정렬)
- MathJax/LaTeX 표준 문법 사용

### 수식 작성 원칙
1. **변수 명명 통일**
   - 스칼라: 소문자 이탤릭 ($x, y, \alpha, \lambda$)
   - 벡터: 소문자 볼드 ($\mathbf{x}, \mathbf{w}$) 또는 화살표 ($\vec{x}$)
   - 행렬: 대문자 볼드 ($\mathbf{A}, \mathbf{W}$) 또는 일반 대문자 ($A, W$)
   - 집합: 대문자 칼리그래피 또는 블랙보드 볼드 ($\mathbb{R}, \mathcal{D}$)

2. **주요 수식 번호**
   - 핵심 정리나 자주 참조할 수식은 번호 부여 고려 (필수는 아님)
   - 번호 형식: 노트 내에서 일관성 유지

3. **기호 설명**
   - 수식 바로 뒤에 각 기호의 의미를 명시
   ```
   $$
   f(x) = Wx + b
   $$
   - $W \in \mathbb{R}^{m \times n}$: 가중치 행렬
   - $x \in \mathbb{R}^n$: 입력 벡터
   - $b \in \mathbb{R}^m$: 편향 벡터
   ```

4. **수식 정렬**
   - 여러 줄 전개는 `\begin{align}` 또는 `\begin{aligned}` 사용
   - 등호 기준 정렬 (`&=`)
   ```
   $$
   \begin{aligned}
   f(x) &= (x + 1)^2 \\
        &= x^2 + 2x + 1
   \end{aligned}
   $$
   ```

5. **행렬 표기**
   ```
   $$
   A = \begin{bmatrix}
   a_{11} & a_{12} \\
   a_{21} & a_{22}
   \end{bmatrix}
   $$
   ```

### 자주 사용하는 수식 기호
- 그래디언트: `\nabla f`, `\frac{\partial f}{\partial x}`
- 노름: `\|x\|`, `\|x\|_2`, `\|x\|_F`
- 전치: `A^\top` (T보다 \top 선호)
- 합: `\sum_{i=1}^{n}`, 곱: `\prod_{i=1}^{n}`
- 기댓값: `\mathbb{E}[X]`, 확률: `P(X=x)`, `p(x)`
- 조건부: `P(Y|X)`, `p(y|x)`
- 정의: `\triangleq` 또는 `:=`
- 근사: `\approx`, 비례: `\propto`

## 코드 작성 규칙

### 코드 블록 기본 형식
- 언어 지정: ` ```python `
- 실행 가능한 완전한 코드 제공
- import 문 명시
- 주석은 "무엇을", "왜"를 설명

### NumPy/PyTorch 예제
- 재현 가능하도록 시드 설정 고려 (`torch.manual_seed(42)`)
- 텐서 형상(shape)을 주석으로 표시
- 작은 예제로 시작 (3x3 행렬 등)
- `dtype` 명시 (특히 수치 정밀도가 중요한 경우)

```python
import torch

# 작은 예제로 개념 설명
A = torch.tensor([[1., 2.],
                  [3., 4.]], dtype=torch.float64)  # shape: (2, 2)

# 연산 수행
eigenvalues, eigenvectors = torch.linalg.eig(A)
print("고유값:", eigenvalues)
```
