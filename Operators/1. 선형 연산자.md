## TL;DR
- 선형 연산자는 입력 벡터 공간에서 출력 벡터 공간으로의 선형 변환을 표현하며, 대부분의 딥러닝 레이어는 선형 연산 + 비선형 활성화로 구성된다.
- 선형성(가법성, 스칼라 배분), 행렬 표현, 고유값/특이값, 자기수반성 등의 개념은 모델 안정성·표현력 분석의 기초가 된다.
- 컨볼루션, 그래프 라플라시안 등 복잡해 보이는 연산도 적절한 기저를 택하면 선형 연산으로 해석할 수 있다.

## 핵심 개념
- **선형성의 두 가지 조건**
  - 가법성(additivity): `T(u + v) = T(u) + T(v)`
  - 동차성(homogeneity): `T(αu) = αT(u)`
  - 두 조건을 만족하면 모든 선형 조합에 대해 `T(αu + βv) = αT(u) + βT(v)`가 성립한다.
- **행렬 표현**
  - 적절한 기저 `{e_i}`를 잡으면 선형 연산자 T는 행렬 A로 표현되어 `T(x) = Ax`.
  - 기저를 바꾸면 행렬 표현은 달라지지만 연산 자체는 동일.
- **고유값과 고유벡터**
  - `T(v) = λv`를 만족하는 (λ, v)는 변환의 “고정된 축”.
  - 대각화 가능할 경우, 행렬은 고유공간의 기저에서 대각 행렬이 된다.
- **Self-adjoint(자기수반) 연산자**
  - 실수 공간에서는 대칭 행렬, `A = A^T`. 고유값이 모두 실수이고 직교 기저를 갖는다.
  - 최적화에서 Hessian이 양의 정부이면 최소점의 존재와 안정성을 보장.
- **Positive definite**
  - 모든 `x ≠ 0`에 대해 `x^T A x > 0` → 역행렬 존재, 에너지 함수로 해석 가능.
- **프로젝션 연산자**
  - `P^2 = P`를 만족. 어떤 부분공간으로의 직교 투영.
  - PCA에서 주성분 subspace로 데이터를 투영하는 과정이 대표적.
- **정규 연산자**
  - `A^*A = AA^*` (복소수 공간). 유니터리/직교 행렬은 길이와 각도 보존.

## PyTorch 실습
```python
import torch

def is_linear(T, x, y, alpha):
    left = T(alpha * x + y)
    right = alpha * T(x) + T(y)
    return torch.allclose(left, right)

A = torch.tensor([[2., 1.],
                  [0., 3.]])
linear_map = lambda v: A @ v
x = torch.tensor([1., 2.])
y = torch.tensor([-1., 0.5])
print("is linear:", is_linear(linear_map, x, y, 0.3))

# 자기수반 여부 확인
print("A == A^T ?", torch.allclose(A, A.t()))

# 고유값
eigvals, eigvecs = torch.linalg.eig(A)
print("eigenvalues:", eigvals)
```
- `is_linear` 함수로 선형성 조건을 직접 검증할 수 있으며, PyTorch의 `torch.linalg.eig`로 고유값 분석이 가능하다.

## 실습 포인트
- 랜덤 행렬을 생성해 자기수반 여부(`torch.allclose(A, A.t())`), positive definiteness (`torch.all(torch.linalg.eigvals(A).real > 0)`)를 검사.
- 프로젝션 행렬 `P = V(V^T V)^{-1}V^T`를 만들어 임의의 벡터를 subspace에 투영해 본다.
- 그래프 라플라시안 `L = D - A`의 고유값을 계산해 그래프 커뮤니티 수와 연결성을 파악.
