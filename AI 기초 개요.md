## TL;DR
- AI 기초 학습은 수학 → 프로그래밍 → 머신러닝 개념 순으로 다지고, 이후 PyTorch와 같은 프레임워크 학습에 연결합니다.
- 각 단계마다 필수 개념과 추천 학습 경로, 참고 노트를 정리해 빠르게 부족한 기반을 보완할 수 있도록 구성했습니다.
- 모든 노트는 동일한 템플릿(개요 → 핵심 개념 → 실습 포인트 → 다음 학습)으로 채워 나갈 예정입니다.

## 학습 전략
- **Layered Approach**: 모르는 개념이 나오면 해당 단계 노트로 돌아가 요약을 확인하고, 필요 시 외부 자료를 추가로 링크합니다.
- **실습 병행**: 수학 노트에서도 간단한 코드 실습(NumPy/PyTorch)으로 감각을 익히고, ML 노트에서는 미니 프로젝트/노트북을 제안합니다.
- **용어 정리**: 각 노트 말미에 용어 사전을 추가해 반복되는 용어를 빠르게 조회할 수 있게 합니다.
